# 《始终允许传送》模组

## 🎯 模组特点
**无需作弊，安全传送** - 专门为联机服务器设计的传送模组，让普通玩家也能安全传送，同时防止滥用！

## 🚀 核心功能
1. **免作弊传送** - 生存模式下也能用传送指令
2. **强化安全** - 自动避开岩浆、悬崖等危险位置
3. **跨维度支持** - 主世界、下界、末地随便传
4. **防滥用机制** - 冷却时间防止乱传送

## 📋 主要指令

### 🔄 玩家传送
| 指令 | 功能 | 例子 |
|------|------|------|
| `/tps <玩家名>` | 传送到其他玩家身边 | `/tps 小明` |
| `/tpshome` | 回自己的重生点 | `/tpshome` |
| `/tpshere <玩家名>` | 让玩家传到你身边 | `/tpshere 小明` |
| `/tps <玩家1> <玩家2>` | 把玩家1传到玩家2那 | `/tps 小明 小红` |

### ⚠️ 重要限制
- **坐标传送** 仅作弊模式可用：`/tps <玩家> x y z`
- **冷却时间** 默认3秒（可配置）
- **安全第一** 危险位置会传送失败

## 🎮 使用场景
- **朋友联机** - 快速集合打怪/盖房子
- **团队协作** - 队长召集队员
- **危险救援** - 把卡住的队友传回家
- **探索辅助** - 累了直接回家

## ⚙️ 可配置项
```toml
# 配置文件 alwaysallowtp-common.toml
冷却时间 = 3      # 0=无冷却
安全搜索范围 = 8   # 危险时找安全点的范围
跨维度传送 = true  # 是否允许跨维度
严格安全检查 = true # 是否开启全面安全检查
```

## 💡 为什么选择这个模组？
- ✅ **安全** - 不会掉进岩浆摔死
- ✅ **方便** - 几个简单指令搞定所有传送需求
- ✅ **公平** - 不破坏游戏平衡
- ✅ **稳定** - 不拖慢服务器
